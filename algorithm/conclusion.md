# 数据的预处理（属性值的规范化）

- 一致化处理：属性值越大，性能越优
- 无量纲化：消除单位对属性值数值大小的影响
- 归一化：把数值映射到区间[0,1]内

常见的数据的处理方法：

- 0-1归一化：$x' = \frac{(x - \min(x))}{\max(x) - \min(x)}$
- 标准化（Z-score）：$x' = x \cdot \sigma + \mu$
- 区间属性变换
- 向量规范化：$x' = \frac{x}{\|x\|_2} = \frac{x}{\sqrt{\sum_{i=1}^{n}x_i^2}}$

# 插值

插值要求强制穿过所有给出的数据点，追求局部精准，**用于补全缺失值**

# 拟合

拟合的实质是最小化整体误差，即找到一条曲线，让所有数据点到曲线的整体误差最小
**数据拟合的关键是选取适当的拟合函数**
大多数情况下，问题的机理是不清楚的，无法得到变量之间的函数关系
所以通常做出数据的散点图，从直观上判断拟合函数
